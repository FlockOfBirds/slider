# Slider
Displays a value on a slider.

## Features
* Show or change a value on a slider
* Execute a microflow when a slider value is changed or clicked
* Show or hide a tooltip on hover

## Dependencies
Mendix 6

## Demo project
https://slider.mxapps.io/
